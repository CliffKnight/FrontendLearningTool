## JS 中的多维数组

在 JS 中是不能直接定义多维数组的，但是可以通过变通的方式来实现多维数组。

**定义方法**

方法一

定义一个 2*3 的数组，并初始化。

```js
let arr = new Array();
for (let i = 0; i < 2; i++) {
  arr[i] = new Array();

    arr[i][j] = i * j;
  }
}
```

方法二

```js
let arr = [[1, 2, 3], [4, 5, 6]];

arr[0][0] // 1
```

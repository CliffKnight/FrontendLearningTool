3. `if` 语句能不能写成 `&&`，`||` 等形式

   ```js
   if(x === 0 ){
     console.log("happyCoding1024");
   }
   ```

   为什么不写成这样呢？

   ```js
   (x === 0) && (console.log("happyCoding1024"));
   ```
